**Step 2: Add handleCancelWorkout function**

Add this function near the other handlers (before `handleFinish`):

```tsx
function handleCancelWorkout() {
  Alert.alert(
    'Cancel Workout',
    'Discard this workout? All progress will be lost.',
    [
      { text: 'Keep Going', style: 'cancel' },
      {
        text: 'Discard',
        style: 'destructive',
        onPress: async () => {
          const workout = workoutRef.current;
          if (workout) {
            await deleteWorkout(workout.id);
          }
          if (timerRef.current) clearInterval(timerRef.current);
          dismissRest();
          setActiveWorkout(null);
          workoutRef.current = null;
          setExerciseBlocks([]);
          setUpcomingTargets(null);
          loadState();
        },
      },
    ],
  );
}
```

Add `Alert` to the react-native imports and `deleteWorkout` to the database imports (check if already imported — it's defined in database.ts but may not be imported in WorkoutScreen).

**Step 3: Add padding between header elements**
